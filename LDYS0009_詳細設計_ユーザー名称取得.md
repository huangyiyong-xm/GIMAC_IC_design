## 1. 処理概要

### 1.1. 機能概要

ユーザIDから、ACCESの情報を元にユーザの名称を求める。
引数で、英名を返すか日本名を返すかを指定することができる。

## 2. 詳細処理

### 2.1. 引数の取得とチェック

- 引数.ユーザーＩＤ が　ブランク　又は　 NULL 　の場合、エラーメッセージを出力し処理終了。
  - エラーコード : E.LDP10917
  - エラーメッセージ : 'Specify the User ID.'
  - (ユーザーＩＤを指定してください。)

### 2.2. 初期処理

利用する変数を初期化する。

| No. | 変数論理名      | 初期化設定値 |
| :-: | --------------- | ------------ |
|  1  | 変数.ユーザ名称 | スペース     |

### 2.3. 主処理

#### 2.3.1.ユーザー名称取得

- 引数.ロケーション = '01' の場合
  ユーザマスタを検索する
  ```sql
  SELECT 氏名現地語
    INTO 変数.ユーザ名称
    FROM ユーザマスタ a
   WHERE a.ユーザID = 引数.ユーザーID
  ```
- それ以外の場合
  ユーザマスタを検索する
  ```sql
  SELECT 氏名英語
    INTO 変数.ユーザ名称
    FROM ユーザマスタ a
   WHERE a.ユーザID = 引数.ユーザーID
  ```

### 2.4. 終了処理

取得したユーザ名称を戻り値に設定する。

| 戻り値論理名     | 設定値          |
| ---------------- | --------------- |
| 処理ステータス   | 0               |
| SQLコード        | スペース        |
| エラーコード     | スペース        |
| エラーメッセージ | スペース        |
| エラー位置       | スペース        |
| ユーザ名称       | 変数.ユーザ名称 |

## 3. 補足説明

### 3.1. 戻り値について

- ステータスについて
  - 0 : Normal End
  - -1 : Abnormal End
  - -2 : PGM Error

### 3.2. エラー発生時の対応について
- 戻り値.エラー位置 :'LDYS0009'
- SQLエラーが発生した場合、エラーログを出力して処理終了する
