#### 2.3.4 在庫ファイル更新

変数.オーダーステータス = '2'　の場合
    品目マスターを検索する
```sql
SELECT  品目ステータス
  into  変数.品目ステータス
FROM  品目マスター
 WHERE 品目番号    = 引数.子品目番号
   AND 供給者      = 引数.子供給者 
   AND 使用者      = 引数.子使用者
```
    データが存在しない場合
        変数.品目ステータス = ''

    MRP情報値を検索する
```sql
SELECT  MRP需要方針コード
  into  変数.MRP需要方針コード
FROM  MRP情報値
 WHERE 品目番号    = 引数.子品目番号
   AND 供給者      = 引数.子供給者 
   AND 使用者      = 引数.子使用者
```
    データが存在しない場合
        変数.MRP需要方針コード = ''

    変数.発注済所要数(従属所要)　=  変数.所要数 - 変数.出庫数

    変数.構成品サイン = '-' かつ　 変数.発注済所要数(従属所要)　> 0　の場合
        変数.発注済所要数(従属所要)　=  0

    変数.構成品サイン <> '-' かつ　 変数.発注済所要数(従属所要)　< 0　の場合
        変数.発注済所要数(従属所要)　=  0

    CALL sp 在庫ファイル更新
