### 2.3. 主処理

#### 2.3.1. 工場処理日存在チェック

IC工場処理日を検索する

```sql
  SELECT IC工場処理日,
         基点システム日時
    INTO 変数.IC工場処理日,
         変数.基点システム日時
    FROM IC工場処理日
   WHERE 処理タイプ = 引数.処理ID 
```

- データが存在しない場合、エラーメッセージを出力し処理終了。
  - エラーコード：'LDP10911';
  - エラーメッセージ：'The IC pymac date is not exist.'
    - (IC工場処理日が存在しません。)

#### 2.3.2. 工場処理日設定

- 処理モード = 'A' or 'a'の時  ※基点システム日時を、システム日付にて変更する

  ```sql
  UPDATE IC工場処理日 
    SET  基点システム日時 = TO_CHAR(変数.システム日付,"%Y%m%d"),
         更新者 = 'LDYS0011',
         更新カウンター = 更新カウンター + 1,
         更新日時 = 変数.システム日付,
         更新pgmid = 'LDYS0011'
   WHERE 処理タイプ = 引数.処理ID 
  ```
- 処理モード ='B' or 'b'の時   ※IC工場処理日を１日加算する。
  変数.new工場処理日 = TO_DATE(変数.基点システム日時,"%Y%m%d")
  変数.new工場処理日 = 変数.new工場処理日 + interval(1) day to day

  - 変数.new工場処理日 >= 変数.IC工場処理日の時
    IC工場処理日を更新する

    ```sql
    UPDATE IC工場処理日 
      SET  IC工場処理日 = TO_CHAR(変数.new工場処理日),
           更新者 = 'LDYS0011',
           更新カウンター = 更新カウンター + 1,
           更新日時 = 変数.システム日付,
           更新pgmid = 'LDYS0011'
     WHERE 処理タイプ = 引数.処理ID 
    ```
  - それ以外の時、エラーメッセージを出力し処理終了。

    - エラーコード：'W001';
    - エラーメッセージ："Not Update :: (New pymac_date [" || trim(変数.new工場処理日) || "]) < (Current pymac_date [" || trim(変数.IC工場処理日)   || "])"
- 処理モード ='C' or 'c'の時

  - 変数.基点システム日時[7,8] > 15 の時     ※工場処理日を翌月１日に変更する。

    - 変数.基点システム日時 = 変数.基点システム日時[1,6] || "01";
      変数.new工場処理日 = TO_DATE(変数.基点システム日時,"%Y%m%d")
      変数.new工場処理日 = 変数.new工場処理日 + interval(1)  month to month
  - それ以外の時 （変数.基点システム日時が15日よりも以前の場合） ※変数.new工場処理日 を当月の1日に設定する

    - 変数.new工場処理日 = TO_DATE(変数.基点システム日時,"%Y%m%d")
      変数.new工場処理日 = TO_DATE(変数.new工場処理日,"%Y%m%01")
  - 変数.new工場処理日 > 変数.IC工場処理日 の時
    IC工場処理日を更新する

    ```sql
    UPDATE IC工場処理日 
      SET  IC工場処理日 = TO_CHAR(変数.new工場処理日),
           更新者 = 'LDYS0011',
           更新カウンター = 更新カウンター + 1,
           更新日時 = 変数.システム日付,
           更新pgmid = 'LDYS0011'
     WHERE 処理タイプ = 引数.処理ID 
    ```
  - それ以外の時、エラーメッセージを出力し処理終了。

    - エラーコード：'W001';
    - エラーメッセージ："Not Update :: (New pymac_date [" || trim(変数.new工場処理日) || "]) < (Current pymac_date [" || trim(変数.IC工場処理日)   || "])"
- 処理モード ='D' or 'd'の時   ※工場処理日をシステム日付に変更する。
  変数.new工場処理日 = TO_DATE(変数.システム日付,"%Y%m%d")

  - 変数.new工場処理日 >= 変数.IC工場処理日 の時
    IC工場処理日を更新する
    ```sql
    UPDATE IC工場処理日 
      SET  IC工場処理日 = TO_CHAR(変数.new工場処理日),
           更新者 = 'LDYS0011',
           更新カウンター = 更新カウンター + 1,
           更新日時 = 変数.システム日付,
           更新pgmid = 'LDYS0011'
     WHERE 処理タイプ = 引数.処理ID 
    ```
  - それ以外の時、エラーメッセージを出力し処理終了。
    - エラーコード：'W001';
    - エラーメッセージ："Not Update :: (New pymac_date [" || trim(変数.new工場処理日) || "]) < (Current pymac_date [" || trim(変数.IC工場処理日)   || "])"
- length(引数.処理モード) = 8の時 (処理モードの桁数が8の時)  ※処理モードのパラメータ日付で更新をする
  変数.new工場処理日 = TO_DATE(引数.処理モード,"%Y%m%d")

  ```sql
  UPDATE IC工場処理日 
    SET  IC工場処理日, = TO_CHAR(変数.new工場処理日),
         基点システム日時 = TO_CHAR(変数.new工場処理日),
         更新者 = 'LDYS0011',
         更新カウンター = 更新カウンター + 1,
         更新日時 = 変数.システム日付,
         更新pgmid = 'LDYS0011'
   WHERE 処理タイプ = 引数.処理ID 
  ```
- それ以外の時、エラーメッセージを出力し処理終了。

  - エラーコード：'E000';
  - エラーメッセージ："Argument Error.  : Arg1 = [" || trim(引数.処理モード) || "]"

## 3. 補足説明

### 3.1. 戻り値について

- ステータスについて
  - 0 : Normal End
  - -1 : Abnormal End
  - -2 : PGM Error

### 3.2. エラー発生時の対応について

- 戻り値.エラー位置 :'LDYS0011'
- SQLエラーが発生した場合、エラーログを出力して処理終了する
