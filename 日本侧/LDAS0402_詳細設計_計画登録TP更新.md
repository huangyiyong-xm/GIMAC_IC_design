## 2. 詳細処理

### 2.1. 引数の取得とチェック

特記無し

### 2.2. 初期処理

- システム日時セット

```sql
　変数.システム日時 := システム日時
```

### 2.3. 主処理

エラーレベルステータス の値に応じて、計画登録_トランザクションプール (le_trn_planning_tp) テーブルの特定レコードに対して更新処理を行う。

1. エラーレベルステータスが空文字の場合
   引数.オーダー番号 が空でない場合に限り、以下の更新を行う：
   オーダー番号 を引数値で更新
   その他、共通で 更新カウンタ、更新日時、更新PGID を更新

```sql
　if エラーレベルステータス = "" then
  if trim(引数.オーダー番号) <> "" then
     update 計画登録_トランザクションプール
        set オーダー番号 = 引数.オーダー番号,
            更新カウンタ = 更新カウンタ ＋ 1,
            更新日時 = 変数.システム日時,
            更新PGID = 'LDAS0402'
      where TP処理番号 = 引数.TP処理番号
        and TP処理明細番号 = 引数.TP処理明細番号
        and 入力区分 = 引数.入力区分;
  end if
```

- trim(引数.オーダー番号)は、引数として渡された「オーダー番号」の前後の空白を取り除く処理です。
- NEXTの通算オーダー番号 (total_orderno) のカラムをオーダー番号(order_no)に変更

2. エラーレベルステータスが "9"  (エラー) 以外の場合
   処理状況 を "1"  (完了) に設定
   エラーレベルステータス を引数値で更新
   その他、共通で 更新カウンタ、更新日時、更新PGID を更新

```sql
　else 
   if エラーレベルステータス <> "9" then
      update 計画登録_トランザクションプール
         set 処理状況 = "1",
             エラーレベルステータス = 引数.エラーレベルステータス,
             更新カウンタ = 更新カウンタ ＋ 1,
             更新日時 = 変数.システム日時,
             更新PGID = 'LDAS0402'
      where  TP処理番号 = 引数.TP処理番号
        and  TP処理明細番号 = 引数.TP処理明細番号
        and  入力区分 = 引数.入力区分;
```

3. エラーレベルステータスが "9" (エラー) の場合
   エラーレベルステータス のみを引数値で更新
   その他、共通で 更新カウンタ、更新日時, 更新PGID を更新

```sql
　   else
      update 計画登録_トランザクションプール
         set エラーレベルステータス = 引数.エラーレベルステータス,
             更新カウンタ = 更新カウンタ ＋ 1,
             更新日時 = 変数.システム日時,
             更新PGID = 'LDAS0402'
       where TP処理番号 = 引数.TP処理番号
         and TP処理明細番号 = 引数.TP処理明細番号
         and 入力区分 = 引数.入力区分;
   end if
end if
```

## 3. 補足説明

### 3.1. 戻り値について

- ステータスについて
  - 0 : Normal End
  - -1 : Abnormal End
  - -2 : PGM エラー

### 3.2. エラー発生時の対応について

- SQL エラーが発生した場合、エラーログを出力して処理終了
  | No. | 戻り値           | 属性    | 設定値     |
  | --- | ---------------- | ------- | ---------- |
  | 1   | 処理ステータス   | INTEGER | -1         |
  | 2   | SQL コード       | VARCHAR | SQLSTATE   |
  | 3   | エラーコード     | VARCHAR | スペース   |
  | 4   | エラーメッセージ | VARCHAR | SQLERRM    |
  | 5   | エラー位置       | VARCHAR | 'LDAS0402' |
